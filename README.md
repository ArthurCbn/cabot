# Cabot

## Dependencies

```
ffmpeg
```

## Installation

```
git clone https://ArthurCbn:**key**@github.com/ArthurCbn/cabot.git
cd cabot
bash installation.sh
source ~/.bashrc
```

## Usage

One command for one feature : updating all your playlists (download in lossless, convert, analyse key) 

```
cabot
```
